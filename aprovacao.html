O que é um Carrossel?

Um carrossel no LinkedIn é uma sequência de imagens (como slides de apresentação) que aparecem uma de cada vez no post. A pessoa desliza para o lado no celular ou clica na seta no computador para ver o próximo slide. No LinkedIn, aparece uma imagem grande de cada vez — não todas juntas.
